Profile Template, Advanced Tab

Use the Advanced tab of the profile template to define the security and format for exchanging messages with the partner.

The My Profile template does not contain an Advanced tab.

Use this To do this

Insert signature certificate

Click to insert a digital signature certificate issued from companies that are official certification authorities (CA).

Note:
To remove a signature certificate, right-click the signature certificate section, and select Remove signature certificate.

Signature certificate

Click the ellipsis button […] to select a signature certificate. In the Select Certificate dialog box, select the certificate, and then click OK.

Insert global port

Click to associate a send port with the profile. You can select an existing send port, or configure a new send port.

Note:
To remove a global port, right-click the global port section, and select Remove global port.

Global port name

Type a friendly name for the port.

Note:
Required fields are marked with a red asterisk (*) and are underlined. All other fields are optional.

Associating a send port with the profile

You can choose either to select an existing send port for the profile, or to configure a new send port for the profile.

To select an existing port

  1. On the Partner Profile, Advanced tab, click the Choose BizTalk send port ellipsis button […].

  2. In the Select Mapped Port dialog box, from the Select send port list, select the port, and then click OK.

    Note:
    To instead configure a new port, right-click the Choose BizTalk send port section, and then click Specify port configuration.

  3. Click the Submit Profile tab to save the data to the server.
To configure a new port

  1. On the Partner Profile, Advanced tab, right-click Choose BizTalk send port, and then click Specify port configuration.

    Note:
    If you associated an existing port with the profile, and then for the same global port choose to configure a new port, BAS removes the existing port you selected.

  2. See the following for assistance in configuring a new send port:

    • To configure an HTTP send port, follow the instructions in How to Configure an HTTP Send Port.

    • To configure a FILE send port, follow the instructions in How to Configure a FILE Send Port.

    • To configure a SOAP send port, follow the instructions in How to Configure a SOAP Send Port.

      Note:
      To instead select an existing port, right-click the configure port section, and then select Choose existing port.

  3. Click the Submit Profile tab to save the data to the server.
