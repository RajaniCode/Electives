3.3 Element Selector Shortcuts

Use the $() shortcut to find all DOM elements with an ID of "#calendar" in the sample app.

Use the inspect() function and the $() shortcut to view the first element that has a class of ".event" in the Elements panel. Then change that event name to Run 10K.

Now that we're in the Elements panel, click to select the Build Robot event. Then, back in the console, use the $0 shortcut and a DOM method to remove() it from the list.
