5.2 New Project

Let's use mix to create a new project.

Using the mix tool, create a new project called expenses.


Type help to see the help menu.

Press ENTER to submit your answer.


> mix new expenses
* creating README.md
* creating .gitignore
* creating mix.exs
* creating config
* creating config/config.exs
* creating lib
* creating lib/expenses.ex
* creating test
* creating test/test_helper.exs
* creating test/expenses_test.exs

Your Mix project was created successfully.
Success!
