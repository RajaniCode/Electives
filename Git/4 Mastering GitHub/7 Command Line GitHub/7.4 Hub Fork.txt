7.4 Hub Fork

Now that we have the repo cloned, go ahead and fork it using hub (or git).


Type help to see the help menu

$ alias git=hub
$ git clone deadlyvipers/dojo_rules
$ cd dojo_rules


$ hub fork

Updating repo
From ssh://github.com/deadlyvipers/dojo_rules
 * [new branch]      gh-pages    -> yourusername/gh-pages
 * [new branch]      master      -> yourusername/master
new remote: yourusername

Congratulations, you're correct!
