1.9 A Pirate's Favorite Parameter

When using the os.Args array from a program run with the following command, which value is assigned to os.Args[0]?

$ go run main.go "192.168.1.0"

[Single Answer]

The value 192.168.1.0.
The name of the operating system.
The name of the temporary executable file.
An empty string.


Answer: The name of the temporary executable file.
