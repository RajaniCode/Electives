2.3 Declaring Type Manually

Which of the different options below represents the manual type declaration alternative to the following variable assignment: listenPort := "8080"?

[Single Answer]

var listenPort int
listenPort = "8080"

var listenPort string
listenPort = "8080"

var string listenPort
listenPort = "8080"


Answer:
var listenPort string
listenPort = "8080"
