2.11 Exit Codes

Exit codes are used by other programs so they know whether or not an error occurred. The exit code 1 is a POSIX standard indicating which of the following?

[Single Answer]

The program has finished with success.
The program has finished, but errors have occurred.
The compiler is still running.
Missing a semicolon at the end of the line.


Answer: The program has finished, but errors have occurred.

Note: 
The exit code 1 does not mean success.
The exit code does not relate to the compiler.
The Go programming language does not require semicolons at the end of a line.
