
Shift+Ctrl+/ to Add Block Comment in both XML and JSP files (works at the line level or for a selected block). 


Shift+Ctrl+\ to Remove Block Comment 


Just in case, if you are looking for a particular shortcut, you can use Shift+Ctrl+L to Show Key Assist (i.e. the Commands and their Bindings).


Single line 

comment Ctrl + /

uncomment Ctrl + /

Multiline

comment Ctrl + Shift + /

uncomment Ctrl + Shift + \ (note the backslash)
