2.5 Shuffle

Which of the following defines the behavior of the shuffle() function?

[Single Answer]

It randomizes the order of the elements in an array.
It picks one or more random entries out of an array and returns the key.
It is used to sort several arrays at once or a multidimensional array by one or more dimensions.


Answer: It randomizes the order of the elements in an array.

Note:
It picks one or more random entries out of an array and returns the key.
This describes array_rand.

It is used to sort several arrays at once or a multidimensional array by one or more dimensions.
This describes array_multisort.
