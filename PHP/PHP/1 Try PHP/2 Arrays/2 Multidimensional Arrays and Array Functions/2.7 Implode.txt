2.7 Implode

Which of the following defines the behavior of the implode() function?

[Single Answer]

Join array elements with a glue string as the first of two arguments.
Returns an array of strings, each of which is a substring of the string formed by splitting it on boundaries formed by the string delimiter.
Returns a string with backslashes stripped off.


Answer: Join array elements with a glue string as the first of two arguments.
