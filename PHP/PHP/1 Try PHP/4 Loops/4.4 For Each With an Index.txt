Create navigation with a foreach loop.

<?php
$cities = array(

    </div>

</html>

3. Finally, assign the value of the country as the name inside of the anchor tag. Your final items will all be different values, but should look like this:

<a href='#' class='tokyo'>Japan</a>
Note: The value in the loop is also an array, where we will access each item by its integer key. For example:

$data[1] // Gives you the 'year established'
