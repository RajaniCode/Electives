1.4 Prepping a Form

Before we can write any PHP, we need to set up an HTML form with the right attributes.

travel.php

<form class="form">
  <div class="form-group">
    <label for="name">Name</label>
    <input type="text" class="form-control" id="name">
  </div>

  <div class="form-group">
    <label for="reason">Reason for Visit</label>
    <input type="text" class="form-control" id="reason">
  </div>

  <button type="submit" class="btn" disabled="disabled">Submit</button>
</form>


1. By default, forms use the GET method. Add an attribute to the <form> tag to set the method to "post".

Answer:
travel.php

<form class="form" method="post">
  <div class="form-group">
    <label for="name">Name</label>
    <input type="text" class="form-control" id="name">
  </div>

  <div class="form-group">
    <label for="reason">Reason for Visit</label>
    <input type="text" class="form-control" id="reason">
  </div>

  <button type="submit" class="btn" disabled="disabled">Submit</button>
</form>

2. Next, we want the the data submitted to this form to feed into travel.php. Add an attribute to the <form> tag to set the action to "travel.php".

Answer:
travel.php

<form class="form" method="post" action="travel.php">
  <div class="form-group">
    <label for="name">Name</label>
    <input type="text" class="form-control" id="name">
  </div>

  <div class="form-group">
    <label for="reason">Reason for Visit</label>
    <input type="text" class="form-control" id="reason">
  </div>

  <button type="submit" class="btn" disabled="disabled">Submit</button>
</form>

3. The form styles look great, but when we hit submit, no POST data is being sent over. To start fixing this, add the name attribute with a value of "name" to the first <input>.

Answer:
travel.php

<form class="form" method="post" action="travel.php">
  <div class="form-group">
    <label for="name">Name</label>
    <input type="text" class="form-control" id="name" name="name">
  </div>

  <div class="form-group">
    <label for="reason">Reason for Visit</label>
    <input type="text" class="form-control" id="reason">
  </div>

  <button type="submit" class="btn" disabled="disabled">Submit</button>
</form>

4. Now, add the name attribute with a value of "reason" to the second <input>.

Answer:
travel.php

<form class="form" method="post" action="travel.php">
  <div class="form-group">
    <label for="name">Name</label>
    <input type="text" class="form-control" id="name" name="name">
  </div>

  <div class="form-group">
    <label for="reason">Reason for Visit</label>
    <input type="text" class="form-control" id="reason" name="reason">
  </div>

  <button type="submit" class="btn" disabled="disabled">Submit</button>
</form>
