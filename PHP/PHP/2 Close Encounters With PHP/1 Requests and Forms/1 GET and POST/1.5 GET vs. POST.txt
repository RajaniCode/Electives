1.5 GET vs. POST

Which of the following are reasons you might choose a POST request over a GET request?

[Multiple Answer]

You want a URL with a query string that you can bookmark.
You want to send data to a resource via a URL request.
You want to make sure your responses aren't cached by the browser.


Answer:
You want to send data to a resource via a URL request.
You want to make sure your responses aren't cached by the browser.
