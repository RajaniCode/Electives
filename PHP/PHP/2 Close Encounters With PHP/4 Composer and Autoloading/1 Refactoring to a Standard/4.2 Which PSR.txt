4.2 Which PSR?

Based on the PHP Standards Recommendations [http://go.codeschool.com/psr], what is the purpose of PSR-2?

[Single Answer]

HTTP Message Interface
Basic Coding Standard
Autoloading Standard
Logger Interface
Coding Style Guide


Answer: Coding Style Guide
Note:
HTTP Message Interface is PSR-7
Basic Coding Standard is PSR-1
Autoloading Standard is PSR-4
Logger Interface is PSR-3
