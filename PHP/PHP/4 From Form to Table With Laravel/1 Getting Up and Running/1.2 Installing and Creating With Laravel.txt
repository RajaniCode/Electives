1.2 Installing and Creating With Laravel

We will be installing Laravel's installer globally using the composer require command, then creating a new application.


1. Using composer, globally require the package laravel/installer.

Answer:
$ composer global require laravel/installer

2. Now that we have the Laravel installer available, create a new project called market.

Answer: laravel new market
