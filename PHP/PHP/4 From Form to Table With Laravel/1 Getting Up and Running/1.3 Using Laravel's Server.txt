1.3 Using Laravel's Server

Using Laravel's artisan, we will run the internal web server. If you're following along locally on your computer, after this you can visit http://localhost and see a preview page to verify the server is working.


Using php artisan, run the command to start the internal server.

Answer
$ php artisan serve


# Note
$ php artisan serve                                                             
Laravel development server started: <http://127.0.0.1:8000>
