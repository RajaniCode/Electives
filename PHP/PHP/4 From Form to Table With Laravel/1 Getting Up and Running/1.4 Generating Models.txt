1.4 Generating Models

In order to work with our farms, we need to create a Farm model with a migration.

The help entry for make:model is displayed for us.


1.  Using the php artisan command line utility, create a model named Farm and pass the -m flag to generate a migration.

Answer
$ php artisan make:model -m Farm
