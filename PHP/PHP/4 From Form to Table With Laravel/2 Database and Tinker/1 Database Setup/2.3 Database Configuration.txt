2.3 Database Configuration

We now have our database created, so it's time to tell our Laravel project how to connect to the database. We will do that by editing the .env, or environment file.

.env
APP_ENV=local
APP_KEY=base64:dtvn6O1xCrh1lkjsapojkmndoinkmkwpomklnfdjbmnsljbR9ENtzruY+s=
APP_DEBUG=true
APP_LOG_LEVEL=debug
APP_URL=http://localhost

DB_CONNECTION=mysql
DB_HOST=mysql
DB_PORT=3306
DB_DATABASE=homestead
DB_USERNAME=homestead
DB_PASSWORD=secret

BROADCAST_DRIVER=log
CACHE_DRIVER=file
SESSION_DRIVER=file
QUEUE_DRIVER=sync

...
